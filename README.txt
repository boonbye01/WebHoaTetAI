WebHoaTetAI (PHP + MySQL)

Setup (local):
1) Create a MySQL database (example: webhoatetai).
2) Update config.php with db name/user/password.
3) Run schema.sql in your database.
4) Put this folder under your local web server (XAMPP/WAMP).
5) Open login.php in the browser and create the first admin user.

Notes:
- Manage flower types first in flowers.php.
- Add or edit customers in index.php.
- The first visit to login.php creates the admin (only when no admin exists).
